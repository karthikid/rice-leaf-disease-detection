# rice-leaf-disease-detection
# 🌾 Rice Leaf Disease Detection System

A web-based Rice Leaf Disease Detection System developed using **Django**, **Python**, and **Deep Learning** techniques to identify diseases in rice plant leaves from uploaded images.

This system helps farmers and agricultural experts detect rice crop diseases quickly and accurately, reducing crop loss and improving productivity.

---

## 📌 Project Overview

Rice crops are highly vulnerable to various diseases that can reduce agricultural yield. Traditional disease identification methods are time-consuming and require expert knowledge.

This project uses **Machine Learning / Deep Learning** to automatically classify rice leaf diseases through image analysis.

Users can upload a rice leaf image, and the system predicts the disease type with confidence and provides preventive measures.

## 🚀 Features

✅ Rice leaf disease detection using image upload

## ⚙️ Installation

### 1️⃣ Clone Repository

```bash
git clone https://github.com/karthikid/rice-leaf-disease-detection.git
```

### 2️⃣ Move into Project Folder

```bash
cd rice-leaf-disease-detection
```

### 3️⃣ Create Virtual Environment

```bash
python -m venv venv
```

### 4️⃣ Activate Virtual Environment

#### Windows

```bash
venv\Scripts\activate
```

#### Linux / Mac

```bash
source venv/bin/activate
```

### 5️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 6️⃣ Run Django Server

```bash
python manage.py runserver
```
